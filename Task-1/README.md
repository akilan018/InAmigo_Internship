## 📌 Project Overview

This is a fully responsive, premium-designed **NGO Awareness Webpage** created as part of the InAmigos Foundation internship program. The webpage spreads awareness about IAF's six ongoing social impact projects, its mission, certifications, and how people can get involved.

Built with **pure HTML, CSS, and JavaScript** — no frameworks, no dependencies beyond Google Fonts and Font Awesome icons.

## 📄 Pages Description

### `index.html` — Main Homepage
The central awareness page with the following sections:

| Section | Description |
|---------|-------------|
| 🔝 **Navbar** | Sticky, responsive navigation with logo & mobile hamburger menu |
| 🦸 **Hero** | Full-screen animated hero with particle effects & live stats counters |
| 🏛️ **About** | IAF founding story, certifications, address |
| 🎯 **Mission & Vision** | 3-card grid covering vision, mission, and values |
| 🛠️ **Projects** | All 6 IAF projects with images, descriptions, hashtags & "Read More" links |
| 💥 **Social Impact** | Why each project matters — narrative cards |
| 📊 **Counter Strip** | Animated counters: followers, volunteers, projects, states |
| 🖼️ **Gallery** | Lightbox photo gallery of all 6 projects |
| 💬 **Testimonials** | Community voices and beneficiary quotes |
| 🙌 **Join Us** | Volunteer / Donate / Spread the Word CTAs |
| 📞 **Contact Us** | Address, email, social links, and interactive contact form |
| 🔚 **Footer** | Logo, quick links, project links, social icons, hashtags |

Each of the 6 project pages contains:

| Section | Description |

| 🧭 **Breadcrumb** | Navigation trail back to home |
| 📖 **Overview** | Detailed "Why this project?" story |
| 🎯 **6 Goals** | Numbered objective cards |
| ⚡ **5 Activities** | What volunteers actually do on the ground |
| 📊 **Impact Metrics** | Animated counters with real statistics |
| 🙌 **How to Help** | 3-card CTA section (Volunteer / Donate / Action) |
| 🔗 **Other Projects** | Quick navigation to all other 5 project pages |
| 🔚 **Footer** | Consistent footer with socials |

## ✨ Features

- ✅ **Fully Responsive** — works on mobile, tablet, and desktop
- ✅ **Animated Number Counters** — smooth count-up animations triggered on scroll
- ✅ **Scroll Reveal Animations** — elements fade in as user scrolls
- ✅ **Floating Particle Hero** — ambient animated particle background
- ✅ **Gallery Lightbox** — click-to-zoom on gallery images
- ✅ **3D Card Tilt Effect** — subtle perspective hover on project cards
- ✅ **Scroll Progress Bar** — orange progress bar on project pages
- ✅ **Contact Form** — animated submit with spinner & success state
- ✅ **Mobile Hamburger Menu** — animated mobile navigation
- ✅ **Back-to-Top Button** — smooth scroll button
- ✅ **Active Nav Highlighting** — navbar highlights current section
- ✅ **Per-Project Accent Colors** — each project has a unique color identity:
  - 📚 Bachpanshala → Amber/Gold
  - 👩 Udaan → Pink/Rose
  - 🌿 Prakriti → Green/Teal
  - 🐾 Jeev → Sky Blue
  - 🍽️ Seva → Orange
  - 🚀 Vikas → Purple

## 🚀 How to Run

This is a **static website** — no server or build tools required.

```bash
# Option 1: Just open in browser
Open index.html in any modern browser (Chrome, Firefox, Edge, Safari)

# Option 2: Run a local server (optional, for best experience)
# Using Python:
python -m http.server 8000
# Then open: http://localhost:8000

# Using VS Code:
Install "Live Server" extension → Right-click index.html → "Open with Live Server"
```
