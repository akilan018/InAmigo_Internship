# AI-Powered Data Analysis Report

## Overview

This project was completed as **Task 2** of the **InAmigos Foundation** internship programme.

**Topic:** How Artificial Intelligence is Transforming Education for Underserved Communities

---

## Task Objective

> Use AI tools to analyze a topic related to social impact, NGOs, volunteering, fundraising, or education and prepare a short report summarizing key findings, trends, statistics, and insights.

---

## Report Contents

| Section | Description |
|---|---|
| Introduction | Background on AI's role in education |
| Key Findings | Personalized learning, accessibility, teacher support, remote learning |
| Statistics & Trends | Market data, engagement stats, global adoption figures |
| Social Impact | Benefits to learners, educators, and NGOs |
| Challenges | Infrastructure, cost, privacy, and localisation barriers |
| Conclusion | Summary and recommendations |
| References | UNESCO, WEF, UNICEF, HolonIQ, EdTech reports |

## AI Tools Used

- **ChatGPT (OpenAI)** — Research synthesis and content drafting
- **Google Gemini** — Trend analysis and statistical cross-referencing
- **Microsoft Copilot** — Editing, language refinement, and formatting suggestions
- **Claude (Anthropic)** — Report structure review and accuracy checking
