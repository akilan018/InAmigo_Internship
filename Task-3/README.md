# 🌍 Volunteer Opportunities, Internships & Fellowships for Students in India

> **InAmigos Foundation Internship — Task 3: Research & Data Organization**

---

## 📋 About This Project

This file was compiled as part of **Task 3** of the **InAmigos Foundation** internship programme.

The goal was to research and organize **18 verified volunteer opportunities, internships, and fellowships** available for college students across India — presented in a clean, structured, and visually appealing Excel format.


## 🗂️ Sheet 1 — Opportunities (Main Table)

| Column | Description |
|---|---|
| # | Serial number |
| Organization | Name of the NGO / Foundation / Programme |
| Type | Volunteering / Internship / Fellowship / Community Service |
| Eligibility | Who can apply (age, degree, year) |
| Duration | Timeframe of the opportunity |
| Mode | Remote / On-site / Hybrid |
| Focus Area | Social cause the opportunity targets |
| Application Link | Direct URL to apply |
| Deadline | Rolling or specific application window |
| Priority | ⭐ Featured / 🔥 High / ✅ Open |

## 💡 How to Use This File

1. Open in **Microsoft Excel** or **Google Sheets**
2. Start with the **📊 Dashboard** sheet for a quick overview
3. Switch to **🌍 Opportunities** for the full data table
4. Click any **Application Link** to go directly to the opportunity page
5. Filter by **Type**, **Mode**, or **Priority** using Excel's built-in filter

---

## 📌 Notes

- All deadlines marked **"Rolling"** accept applications year-round
- Fellowship deadlines (Teach For India, India Fellow) are typically **October – March**
- NSS enrollment is done through your **college coordinator** — not online
- Links were verified at the time of compilation; always check the official website for updates
